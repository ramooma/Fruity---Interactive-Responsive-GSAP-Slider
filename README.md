# 🍓 Fruity - Interactive Responsive GSAP Slider

### A vibrant and highly interactive product landing page featuring advanced GSAP animations and fluid transitions.

## 📌 Project Overview
This project is a modern, visually engaging landing page designed for a fruit-beverage brand ("Fruity"). The core highlight of this project is the integration of advanced frontend animations, delivering a smooth, premium user experience as users switch between different product sections.

It showcases the ability to build high-end, creative user interfaces with custom transition effects that go beyond standard CSS animations.

## 🚀 Key Features
- **Advanced GSAP Animations:** Utilizes **GSAP (GreenSock Animation Platform)** to handle complex layout transitions and product image movements smoothly.
- **Button Wave Effect:** Custom interactive wave animations on navigation buttons (`Prev`/`Next`) to enhance tactile user feedback.
- **Responsive Fluid Layout:** Perfectly optimized to adapt and look stunning across desktop, tablet, and mobile displays.
- **Dynamic Content Switching:** Seamlessly animates typography, labels, and product graphics simultaneously during transitions.

## 🛠️ Tech Stack & Libraries
- **Frontend Foundation:** HTML5, CSS3, JavaScript
- **Animation Engine:** GSAP (GreenSock Animation Platform)
- **Icons:** Font Awesome (v6.5.1)
- **Concepts:** Creative Web Design, UI/UX Micro-interactions, Responsive Web Design (RWD).
